# 数据集介绍
  ## 简介
  NFCorpus 是用于医学信息检索的全文英文检索数据集。它总共包含 3,244 个自然语言查询（用非技术英语编写，从 NutritionFacts.org 网站收集），其中 169,756 个自动提取的相关性判断对 9,964 个医学文档（用复杂的术语繁重的语言编写），主要来自 PubMed。
